# Frontend — HealthBridge Africa

This folder is owned by **Ibukun Oluwafemi** (Nigeria 🇳🇬)

## Setup
```bash
cd client
npx create-react-app .
npm start
```

## Components to build
- `VoiceRecorder` — microphone input, records user speech
- `ChatDisplay` — shows conversation history
- `ResponsePlayer` — plays back TTS audio response
- `LanguageSelector` — choose input language (Pidgin, Oromo, Swahili, English)
