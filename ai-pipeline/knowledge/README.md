# Health Knowledge Base — HealthBridge Africa

This folder is owned by **Peggy Eyram Attah** (Ghana 🇬🇭)

## Purpose
These documents are the RAG knowledge base — the source of truth
that HealthBridge Africa retrieves from when answering health questions.
The AI only answers from what is in this knowledge base.

## Files
- `nigeria-health-facts.md` — Common conditions in Nigeria
- `ethiopia-health-facts.md` — Common conditions in Ethiopia
- `ghana-health-facts.md` — Common conditions in Ghana (Peggy owns this)
- `kenya-health-facts.md` — Common conditions in Kenya
- `who-guidelines.md` — Relevant WHO guidelines
- `sources.md` — All sources used

## Format for each condition
```
## [Condition Name]
**Symptoms:** ...
**When to see a doctor:** ...
**What to do at home:** ...
**Emergency signs (go to hospital immediately):** ...
**Source:** [WHO / Ministry of Health]
```
